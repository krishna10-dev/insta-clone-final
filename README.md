# Instagram Clone (MERN Stack)

Modern Instagram Login Page Clone built with React, Node.js, Express, and MongoDB.

## Features
- Modern Instagram Phone Login UI
- Form Validation
- Password Toggle (Show/Hide)
- Backend Login API
- Responsive Design (Mobile & Desktop)

## Setup Instructions

### Backend
1. Go to `server` folder.
2. Run `npm install`.
3. Make sure MongoDB is running on your machine.
4. Run `npm run dev` to start the backend server on port 5000.

### Frontend
1. Go to `client` folder.
2. Run `npm install`.
3. Run `npm run dev` to start the frontend server on port 3000.

## Tech Stack
- **Frontend**: React, Vite, Vanilla CSS, Axios, React-Icons
- **Backend**: Node.js, Express, Mongoose, BcryptJS, Dotenv
- **Database**: MongoDB
